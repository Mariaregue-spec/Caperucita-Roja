# Caperucita-Roja
Este proyecto consiste en una página web estática desarrollada en HTML5 y CSS, basada en el cuento clásico Caperucita Roja. El sitio presenta la historia de forma clara y visual, utilizando ilustraciones libres de derechos y una estructura diseñada para facilitar su lectura y uso con fines educativos o demostrativos.

Descripción del proyecto

La web ofrece una versión simplificada y organizada del cuento, presentada en una sola página. Su diseño prioriza la legibilidad, la estética y la accesibilidad, integrando ilustraciones que complementan la narración sin distraer de la historia principal.

Características principales

Página única (single-page) desarrollada con HTML5 y CSS.

Uso de ilustraciones libres de derechos.

Diseño responsivo que se adapta a diferentes dispositivos.

Navegación sencilla mediante secciones ancladas.

Código organizado y fácil de mantener.

Tecnologías utilizadas

HTML5 – Estructura y contenido.

CSS – Estilos, diseño y distribución visual.

Cómo visualizar el proyecto

No se requiere instalación de dependencias ni servidor externo.

Opción 1: Acceso directo

Abra el archivo [index.html](https://github.com/Mariaregue-spec/Caperucita-Roja.git) en cualquier navegador moderno.

Ilustraciones

Todas las imágenes utilizadas provienen de bancos de recursos libres de derechos.
